📝 NoteBase - React Notes App

A simple and efficient notes-taking web application built with React, Redux Toolkit, and LocalStorage.
It allows users to create, update, delete, view, and search notes with a clean UI.

🚀 Features
🆕 Create new notes
✏️ Edit existing notes
❌ Delete notes
👀 View full note details
🔍 Search notes by title
💾 Data persistence using LocalStorage
⚡ Fast state management using Redux Toolkit
🔔 Toast notifications for actions
🛠️ Tech Stack
React JS
Redux Toolkit
React Router DOM
React Toastify
LocalStorage (for persistence)
Tailwind CSS (UI styling)
